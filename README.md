# Tab-Animation-Extension

Custom animation that shows current active context when pressing tab key making it easier to observe which element is currently being selected

This is a custom extension so will require developer mdoe for any browser ebcasue all browsers requrie signed extesnions to add custom ones

Use dev mode in extensions for chrome/edge, use nighly or developer version for firefox

Animation inspiration - Blackrock: https://blackrock.tal.net/vx/candidate

Really liked it and thought of making an extension so that I can get the animation on all websites, can modify color and speed and other parameters according to taste.
